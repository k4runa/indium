01 // capabilities

Everything a 2D game needs, in one element.

No plugin hunting. The editor, the runtime and the tooling ship as a single binary.

F·01

Visual Editor

Dockable panels, hierarchy, inspector, content browser — with Unity-style transform gizmos, snapping and multi-select.

F·02

Hot-Reloaded C++

Scripts are plain C++20 compiled to a shared library and reloaded live. No bindings, no glue — the entire engine API at native speed.

F·03

Standalone Export

File → Export Game packages your project with the IndiumPlayer runtime into a game you can ship — no editor attached.

F·04

2D Lighting & Post FX

Soft shadows with penumbra, ambient color, light maps — plus a shader post-processing chain, WYSIWYG in the editor.

F·05

Physics & Triggers

Fixed-step rigidbodies, circle/box colliders, joints, raycasts and overlap queries on a spatial grid, trigger zones with callbacks.

F·06

Narrative Toolkit

Dialogue trees, quests, items, cutscene timelines, a story-state blackboard and save slots — authored as JSON, scripted from C++.

F·07

Tilemaps & Particles

Tile painting with auto collision rects, particle systems, trails, decals, sprite sheets and a state-machine animator.

F·08

2.5D Depth & Parallax

Depth layers with per-layer parallax factors and Y-sorting — fake a camera with perspective without leaving 2D.

02 // scripting

Your gameplay code is compiled, not bound.

Most engines wrap their API in a scripting language. Indium hands you the engine itself.

IND_PROP
Inspector-editable members One macro exposes a field to the editor and serializes it with the scene.
⟳ reload
Hot reload while the editor runs Scripts compile to a shared library; the engine swaps it in and rehydrates your scene.
0 ms
Zero binding overhead Lifecycle hooks are virtual calls. Entity fields are plain structs. That's it.
scripts/Player.cppC++20
#include "IndiumEngine.hpp"
using namespace Indium;

class Player : public NativeScript
{
    IND_PROP(float, speed, 220.0f);  // shows up in the Inspector

    REGISTER_SCRIPT(Player)

    void OnUpdate(float dt) override
    {
        if (InputManager::Get().IsDown("right"))
            entity->position.x += speed * dt;
    }

    void OnTriggerEnter2D(Entity* other) override
    {
        if (other->tag == "Coin") Destroy(other);
    }
};

INDIUM_EXPORT_SCRIPTS(Player)
✓ compiled & reloaded — scene state preserved, 1.2s
